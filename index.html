Summary: Latitude vs. X


The purpose of this project was to analyze how weather changes as you get closer to the equator and in doing so, showcase some of the programming skills we learned along the way. To accomplish this analysis, we used Python to first pull data from the OpenWeatherMap API and then to assemble a dataset with data from over 500 cities.

After assembling the dataset with the Python library, Pandas, we used the Python library, Matplotlib, to plot various aspects of the weather vs. latitude. The various weather patterns we looked at included: temperature, cloudiness, wind speed, and humidity. This site provides the source data and visualizations we created as part of the analysis, as wel as explanations and descriptions of any trends and correlations we observed.
